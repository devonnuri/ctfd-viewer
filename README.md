# ctfd-viewer

> Console viewer for CTFd based CTFs

## Usage

1. Install requirements
  
  ```text
  pip3 install requirements.txt
  ```

2. Make a config file
  
  ```text
  [URL]
  [Username or Email]
  [password]
  ```

  ```text
  https://demo.ctfd.io
  devonnuri
  s4f3_p4ssw0rd
  ```

3. Run main.py
  
  ```text
  python3 main.py
  ```
